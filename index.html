Présentation

Je suis Thomas Devaux, étudiant de 21 ans en première année de BUT Informatique à Grenoble, passionné par le monde de l'informatique depuis mon baccalauréat STMG au lycée Pierre Béghin. Avec des compétences solides en programmation, développement web et bases de données, j'ai déjà participé à des projets concrets, notamment le développement d'une application gestion et la création d'un site web. Mon objectif est de continuer à enrichir mes compétences, notamment en explorant les domaines de l'intelligence artificielle et du développement d'applications mobiles. Je suis ouvert aux opportunités de stages et projets en entreprise pour acquérir une expérience pratique.

Projets


Implémentation d'un besoin client

Par groupe de 2, nous avons réalisé un progamme en JAVA, permettant de trier et de classer des dépeches en fonction de 5 catégories. Pour cela, il fallait attribuer un score pour chaque dépeche. Il a donc fallu établir, dabord, manuellement une liste de mots de référence pour chaque catégorie. Nous avons ensuite amélioré notre code pour qu'il fasse automatique ces listes.

Installation d'un poste de développement

Dans ce projet, j'ai installé une machine virtuelle. Sur cette machine j'ai installé Debian 11 avec KDE/Plasma ainsi qu'un logiciel pour le développement informatique. Je l'ai installé en utilisant le package Flatpack et Snap. J'ai ensuite réalisé une carte mentale afin de le refaire facilement.

Création d'une base de donnée

Par groupe de 2, nous devions développer une base de donnée sur le naufrage du Titanic. Nous avons dabord rassemblé les informations utiles sur les naufragés et le bateau ainsi que les canaux de sauvetage. Ensuite avec l'aide de différentes requétes SQL nous avons montré que notre base était fonctionnelle.

Recueil de besoins

Par Groupe de 4, nous avons rassemblé différentes informations sur une entreprise. Nous les avons ensuite rassemblé sur un site web que nous avons créé. Le site web est à destination d'élèves de 3eme dans le cadre de leurs recherche de stage. Le but étant de leurs présenter l'entreprise et ce qu'elle fait pour l'environnement.
